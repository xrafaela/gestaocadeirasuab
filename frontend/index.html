Como usar o Timer Pomodoro

  1. Escolha a disciplina que você vai estudar
  2. Clique em "Iniciar" para começar uma sessão de 25 minutos
  3. Foque completamente no estudo durante esse tempo
  4. Quando o timer tocar, faça uma pausa de 5 minutos
  5. Após 4 pomodoros, faça uma pausa maior de 15-30 minutos

Benefícios:

  • ✓ Aumenta o foco e concentração
  • ✓ Reduz a procrastinação
  • ✓ Melhora a gestão do tempo
  • ✓ Previne fadiga mental
